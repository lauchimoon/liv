# liv
lauchi's image viewer

## Usage
```sh
$ ./liv <image>
    where <image> format is png, bmp, tga, jpg, qoi
```

## Controls
- Toggling information bar: LShift + H
- Rotation:
    - 90 degrees anti-clockwise: E
    - 90 degrees clockwise: R
- Zooming:
    - Out: Z
    - In: X
- Panning:
    - Left: H
    - Down: J
    - Up: K
    - Right: L
- Toggle antialiasing: A
- Exit: Q
